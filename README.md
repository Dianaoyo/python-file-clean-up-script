# Python File Cleanup Script

## Описание
Простой кроссплатформенный скрипт на Python для автоматического удаления файлов в указанной папке, которые не изменялись в течение заданного количества дней.

## Функции
- Удаляет только файлы, игнорируя подпапки.
- Логирует все свои действия в файл `cleanup_log.txt`.
- Кроссплатформенный (работает на Windows, macOS, Linux).
- Настраиваемый порог дней (если файл не изменялся на протяжении этого количества дней или более, он будет удален).

## Как использовать
1. Убедитесь, что у вас установлен Python.
2. Откройте файл `cleanup.py` и измените переменную `TARGET_DIRECTORY` на путь к вашей папке и переменную DAYS_THRESHOLD на порог дней
3. Запустите скрипт: `python cleanup_old_files.py`
